# Handwritten Digit Recognizer (CNN)

## 📌 Project Overview

This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits using the MNIST dataset.

## 🧠 Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

## 📊 Model Performance

* Accuracy: ~98%

## 📷 Output

The model successfully predicts handwritten digits from test images.

## 🚀 How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. View predictions

## 📁 Dataset

MNIST dataset (built-in TensorFlow dataset)
